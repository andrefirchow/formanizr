#### Features
- fluid & responsive form-grid
- pure HTML/CSS - no graphics (optional icons)
- themeable - formanizr included 2 themes
- valid HTML5 markup
- no JavaScript

```html
<form class="formanizr">
    <!-- Hier kommt der eigentliche Formular-Code -->
</form>
```
